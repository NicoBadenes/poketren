# 🚀 Pokedex App - Proyecto React Native

## 📱 Descripción

En este proyecto vamos a construir una aplicación usando **React Native con Expo** que consume una API externa.

El objetivo es aprender:

- useState (manejo de estado)
- useEffect (efectos / llamadas a APIs)
- fetch (consumo de datos)
- FlatList (listas)
- navegación entre pantallas

---

## 🧰 Tecnologías

- React Native
- Expo
- JavaScript
- API REST (PokéAPI)

---

## ⚙️ Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/cristianprofe/pokeapp.git
```

2. Entrar al proyecto:

```bash
cd pokedex-app
```

3. Instalar dependencias:

```bash
npm install
```

---

## ▶️ Ejecutar el proyecto

```bash
npm start
```

Luego:

- Presionar `w` para abrir en navegador 🌐
- Presionar `a` para Android 📱
- Presionar `i` para iOS 🍏
